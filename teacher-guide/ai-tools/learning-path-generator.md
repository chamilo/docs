# Learning Path Generator

The AI Learning Path Generator helps you create structured learning sequences automatically. You provide a topic and objectives, and the AI generates a learning path outline with sections and content items that you can review and customize.

## Accessing the Learning Path Generator

The Learning Path Generator is available when creating a new learning path, provided that:

1. AI helpers are enabled at the platform level
2. The learning path generator is enabled in the course settings
3. At least one AI text provider is configured

Look for the **AI Generator** section within the learning path creation interface.

## How to Generate a Learning Path

![The AI learning path generator form with fields for topic, objectives, and section count](/.gitbook/assets/ai-learning-path-generator.png)

1. Open the AI Generator form
2. Configure the parameters:
   * **Topic** — The subject matter for the learning path
   * **Number of items** — How many content pages the path should contain
   * **Words count per page** — Roughly how long each page should be
   * **Add test after each page** — When ticked, the AI also creates a short multiple-choice quiz after each page (default 2 questions, capped at 5)
3. Click **Generate**
4. The AI produces a learning path with one document item per page and, optionally, a quiz item after each one

## Reviewing and Customizing

The generated learning path is a **starting point**. You should:

* **Review the structure** — Check that the sections are in a logical order
* **Edit content** — Modify the generated text, add your own materials, and remove irrelevant sections
* **Add resources** — Insert your own documents, exercises, links, and other content
* **Set prerequisites** — Configure which sections must be completed before others
* **Add assessments** — Include exercises at appropriate points to check understanding

## AI-Generated Content Disclosure

Content generated by AI is labeled with a disclosure notice, indicating that it was created using artificial intelligence. This transparency helps learners understand the origin of the material.

## Tips

* **Start broad, then refine** — Generate a high-level structure first, then manually add detail and specific resources
* **Use for course planning** — Even if you don't use the generated content directly, the AI's suggested structure can be a useful planning tool
* **Combine with exercise generation** — After generating a learning path, use the Exercise Generator to create assessments for each section
