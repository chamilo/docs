# Stijlbladen

Zoals vermeld in sectie _Stylesheets_, kunnen de Chamilo-stijlen worden bijgewerkt door een eenvoudige kopie van een van de bestaande stijlen in _web/css/themes/_. In deze directory vindt u een reeks stijlmappen, bijv. _chamilo\_green_, die naar wens kan worden gekopieerd, hernoemd en gewijzigd.
